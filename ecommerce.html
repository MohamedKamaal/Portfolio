Full-Stack E-commerce Web Application

A fully responsive e-commerce web application built with a complete stack of modern technologies. This platform allows users to browse products, manage their cart, and securely checkout. Admins can manage inventory, orders, and user roles through a dedicated dashboard.

The project includes essential e-commerce features such as user authentication, product filtering, search, order processing, and stripe payment integration. It also focuses on a clean UI/UX for a smooth shopping experience.

Built with scalability and performance in mind, this solution follows best practices in REST API design, database optimization, and frontend responsiveness.
