Blog Rest API

full featured blog Django Rest API

Full-Stack E-commerce Web Application

A feature-rich blogging RESTful API built with Django and Django REST Framework. This platform enables users to create and interact with articles, manage profiles, and engage with other bloggers through follows, claps, comments, and bookmarks.

Key features include secure JWT-based authentication, article filtering and search, user engagement actions, and admin moderation tools. The API supports full CRUD operations, pagination, and rate limiting for a scalable and secure experience.

Designed with performance and maintainability in mind, this project emphasizes clean architecture, test coverage with pytest and Factory Boy, and follows REST best practices throughout.
